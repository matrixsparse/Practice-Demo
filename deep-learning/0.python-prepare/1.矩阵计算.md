# 矩阵计算

## 基本概念

* 矩阵：矩阵的数组，即二维数组。其中向量和标量都是矩阵的特例
* 向量：是指1*n或者n*1的矩阵
* 标量：1*1的矩阵
* 数组：N维的数组，是矩阵的延伸

## 特殊矩阵

* 全0全1矩阵
* 单位矩阵
