随着 epoch 的变化而如何变化来判断模型效果

如果你自己设计了模型，就会发现并非始终都清楚网络应该具有多少层级，或每层应该有多少个节点，应该使用多少个epoch

```bash
model = Sequential()
model.add(Dense(100,input_shape=...))
model.add(Dense(50))
model.add(Dense(10))
```
