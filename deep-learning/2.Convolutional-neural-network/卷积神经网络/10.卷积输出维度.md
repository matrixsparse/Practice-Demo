# 卷积输出维度

## 设置

```bash
H = height, W = width, D = depth
```

我们有一个输入维度是 32x32x3 (HxWxD)

20个维度为 8x8x3 (HxWxD) 的滤波器

高和宽的stride（步长）都为 2。(S)

padding 大小为1 (P)

>计算新的高度和宽度的公式是

```bash
new_height = (input_height - filter_height + 2 * P)/S + 1
new_width = (input_width - filter_width + 2 * P)/S + 1
```
