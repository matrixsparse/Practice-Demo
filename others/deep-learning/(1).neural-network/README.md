## 第 10 周：迁移学习

深度学习的常见技术之一就是利用预先训练好的网络来解决新问题。例如，你可以使用已经经过巨大数据集训练的卷积网络，在一个小数据集上进行图像分类。这种方法称为迁移学习。本课程中，你将学习如何利用迁移学习对花卉图像进行分类，而无需从头开始训练整个网络。

## 第 12 周：文字嵌入
